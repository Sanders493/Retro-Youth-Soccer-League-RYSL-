# Retro Youth Soccer League (RYSL)

## 🎮 About the Game
Retro Youth Soccer League is a top-down, fast-paced 2D arcade soccer game inspired by Spanish youth football culture. Players lead their team to glory in 5v5 matches.

## 🛠️ Built With

- Unity (2D URP)
- C#
- Unity Input System
- Git + GitHub


## 🚀 How to Run the Project

1. Clone the repo:
   git clone https://github.com/Sanders493/Retro-Youth-Soccer-League-RYSL-
2. Open Unity Hub
3. Click "Add Project"
4. Select the project folder
5. Open with Unity version: Unity 6000.3.7f1 (LTS)
   
## 🎮 Controls

- Move: Arrow Keys
- Pass: S
- Shoot/Take Ball: D
- Sprint: Shift
- Switch Player: A

## 📁 Project Structure

- Assets/Scripts/Core → game systems (GameManager, Audio, etc.)
- Assets/Scripts/Gameplay → players, ball, AI, match logic
- Assets/Scripts/UI → menus and HUD
- Assets/Scripts/Data → team and player data
- Assets/Scripts/Utilities → helpers, constants, and extensions

## 📌 Notes

- Make sure "Force Text Serialization" is enabled in Unity settings
- Use Git branches for features, not main
